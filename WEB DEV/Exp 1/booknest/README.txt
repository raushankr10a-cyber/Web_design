This bundle contains an optimized static Online Bookstore front-end with separate files:
- styles.css  — optimized, responsive stylesheet (use this)
- index.html  — Home / Hero / Featured
- biodata.html — Clean biodata profile
- login.html  — Accessible login form (client-side validation only)
- catalogue.html — Searchable, responsive catalogue

Performance & accessibility improvements included:
- Critical CSS inlined (small) for faster first paint
- Preload stylesheet technique to avoid render-blocking
- Lazy-loaded, responsive images (srcset + sizes)
- Semantic HTML and accessible forms (labels, aria-*)
- Minimal, modern CSS variables and mobile-first layout
- Deferred JS and small footprint client code

Open index.html in a browser to preview.
